Rappels par email

📧 Configuration des rappels

ℹ️ Fonctionnement :
• Indicateur hebdomadaire : 1er rappel le lundi de la semaine suivante + relance le mercredi si non saisi
• Indicateur mensuel : 1er rappel le 1er du mois suivant + relance le 3 si non saisi
• Chaque email contient un lien cliquable vers la page de saisie

⚙️ Configuration requise : Ce système utilise EmailJS (gratuit jusqu'à 200 emails/mois).
1. Créez un compte sur emailjs.com
2. Ajoutez un Email Service (Gmail, Outlook…)
3. Créez un Email Template avec ces variables : {{to_email}}, {{to_name}}, {{subject}}, {{message}}
4. Copiez les identifiants ci-dessous
